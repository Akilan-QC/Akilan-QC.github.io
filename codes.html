Gaussian to XYZ

This is simple python script to extract optimized geometry from gaussian output file and can be stored as XYZ file. It can be coupled with the simple bash script to use it in the HPCs.

Calculating pseudo dihedral angle

A python script to calculate the proper dihedral or pseudo dihedral angle from a PDB file. This code is developed to calculate the pseudo dihedral angle from a base flip DNA during the MD.

GUI for polyrate

A python powered graphical user interface for generating polyrate input files from gaussian output. This GUI is written for beginners to generate simple polyrate input files.
